## 🚀 SwiftDo - Task Management Redefined

A minimalist, high-contrast Todo application built with React and Tailwind CSS.

### 🌟 Features
- **Persistent Storage**: Never lose your tasks; they are saved in your browser.
- **Advanced Filtering**: Sort tasks by their status with a single click.
- **Premium UI**: Dark-mode optimized with a focus on readability and typography.

### 🛠 Tech Stack
- **Framework**: React
- **Styling**: Tailwind CSS
- **Icons**: Heroicons
- **Deployment**: Vercel
